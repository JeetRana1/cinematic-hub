‚è±Ô∏è Resume Modal Flow Timeline

1. Page Loads

Player page loads with player-continue-watching.js

// Detects if this is a reload const isPageReload = sessionStorage.getItem('player_page_loaded');

2. Check for Saved Progress

System queries localStorage for saved viewing progress

3. Evaluate Conditions

Determines if modal should be shown

Threshold: 10 seconds Prevents spam for short views

4. Wait for Video Ready

Ensures video element has loaded metadata

if (video.readyState >= 1 && video.duration > 0) { // Video is ready, show modal }

5. Pause Video & Show Modal

Pauses playback and displays the resume prompt

video.pause(); window.__resumePromptActive = true; showEnhancedResumePrompt(resumeTime, savedProgress, video, movieData);
Modal Appears!

6. User Interaction

User chooses to resume or start over

  • Option A: Resume
    Sets video.currentTime = savedProgress.currentTime
    Continues from saved position
  • Option B: Start Over
    Sets video.currentTime = 0
    Starts from beginning

7. Cleanup

Removes modal and resumes normal playback

overlay.remove(); window.__resumePromptActive = false; video.play();
